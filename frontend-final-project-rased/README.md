# رصد AI — الواجهة الأمامية (Frontend Angular Standalone)

أهلاً بكم في المستودع البرمجي الخاص بالواجهة الأمامية لمشروع تخرج **رصد AI** المبني باستخدام **Angular 17+ Standalone Components**.

---

## 🏗️ هيكلية المجلدات والتطوير (Frontend Folder Structure)

تم تنظيم المشروع ليكون جاهزاً بالكامل للميزات والأدوار دون تعقيد:

* **`src/app/models/`:** واجهات البيانات (Interfaces) لتحديد شكل الكيانات القادمة من قاعدة البيانات.
* **`src/app/services/`:** الخدمات الأساسية (Services) للتواصل مع الـ Backend والـ APIs.
* **`src/app/guards/`:** الحواجز الأمنية للتحقق من هوية وأدوار المستخدمين (Auth & Roles).
* **`src/app/layouts/`:** الهيكل الخارجي للوحة التحكم (Sidebar & Navbar & Main Layout).
* **`src/app/pages/`:** الشاشات الـ 40 مقسمة بالتفصيل لكل الأدوار المطلوبة في المنصة:
  1. **`shared/`:** شاشات تسجيل الدخول، التسجيل للشركات، 404، 403، وصفحة الهبوط الرئيسية.
  2. **`system-admin/`:** إدارة الشركات والنظام الشامل.
  3. **`owner-admin/`:** لوحة تحكم صاحب العمل والمهام وتقارير المبيعات.
  4. **`accountant/`:** شاشات الفواتير، الاستحقاقات والمحاسبة المالية.
  5. **`sales-manager/`:** شاشات إدارة فريق المبيعات والعملاء والصفقات.
  6. **`employee-manager/`:** شاشات الحضور والانصراف، المهام والإجازات للمرؤوسين.
  7. **`employee/`:** شاشات الموظف الشخصية (المهام، الحضور، الإجازات والبروفايل).
  8. **`sales-rep/`:** شاشات مندوب المبيعات (العملاء، الصفقات والـ AI).

## 🚀 طريقة البدء والتشغيل (Run Instructions)

### 1. تثبيت الحزم (Install Packages)
إذا لم تكن الحزم مثبتة، افتح مجلد `Frontend` في سطر الأوامر وشغل:
```bash
npm install
```

### 2. تشغيل المشروع محلياً (Run Server)
لتشغيل السيرفر المحلي لمشاهدة الموقع، شغل الأمر التالي:
```bash
npm start
```
أو
```bash
ng serve
```
سيفتح الموقع تلقائياً على الرابط: `http://localhost:4200/`.

### 3. بناء وتجميع المشروع (Build for Production)
لتجميع الكود للرفع والإنتاج للتأكد من خلوه من الأخطاء:
```bash
npm run build
```

## 🧭 التوجيه والمسارات (Routing System)

تم ربط كافة المكونات المضافة بنظام الـ **Lazy Loading** المتقدم في ملف المسارات:
🔗 `src/app/app.routes.ts`

يمكنك الدخول إلى لوحة التحكم والتنقل بينها عبر المسار العام `/app/` بعد تسجيل الدخول (على سبيل المثال: `/app/owner/dashboard` أو `/app/sales-rep/deals`).
