# Rasd

**RasdAI** — منصة ذكية لإدارة المبيعات والموظفين والتحليلات بالذكاء الاصطناعي.

An intelligent SaaS platform for sales pipeline tracking, employee task management, meeting scheduling, and smart AI audits.

## Tech Stack
- **Backend**: ASP.NET Core Web API (.NET 8) — Clean Architecture (DAL → BLL → API)
- **Frontend**: Angular 19 — Standalone Components, Signals, RxJS
- **Database**: SQL Server (Entity Framework Core)
- **AI**: Grok AI Integration (CoPilot)
